关于django连接mysql出现django.core.exceptions.ImproperlyConfigured/ mysqlclient 1.3.13 or newer is required; you have 0.9.3.md

发表于 2019-08-21 更新于 2019-08-25 分类于 错误总结

在网上找了一下,很多朋友都是碰到的错误一和三,所以在这里记录一下

  • 1.在python3.7,Django==2.2.4版本下用Django连接到数据库一直出现:

    • django.core.exceptions.ImproperlyConfigured: mysqlclient 1.3.13 or newer is required;

      you have 0.9.3.

  • 解决方法:

    • cd到.virtualenvs/Django04/lib/python3.7/site-packages/django/db/backends/mysql/
    • vim +35 base.py
    • 注释掉35-36行的代码

  • 2.然后又弹出一个错误:

    • RuntimeError: cryptography is required for sha256_password or caching_sha2_password
    • 解决方法:
      • pip3 install cryptography

  • 3.最后一个错误:

    • AttributeError: ‘str’ object has no attribute ‘decode’

    • 解决方法:在上面的路径下vim +146 operations.py

    • 把146行的query = query.decode(errors='replace')改为query = query.encode(errors='replace')
