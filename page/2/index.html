关于Redis数据库的安装和配置

发表于 2019-05-27 更新于 2019-10-10

  • 下载和安装

    • 打开terminal,依次使用以下命令下载,解压,安装

  • 到这里Redis就安装好了,安装的路径在/usr/local/bin/

  • 把Redis的配置文件copy到/etc/redis/下,并修改相应配置

    • sudo cp redis.conf /etc/redis/redis.conf
    • vi /etc/redis/redis.conf
      • 修改绑定的IP地址
      • 添加log日志的路径logfile "/var/log/redis/redis-server.log"
      • 修改daemonize no为daemonize yes设置为守护进程
      • 其余的配置可以按需修改

  • 将接下来就可以启动Redis的服务端了

    • sudo redis-server /etc/redis/redis.conf

  • 启动服务端

    • redis-cli
